Instructions
To use the Maquette Generator, please follow these instructions:
  1. Fill with placeholders or Import codes from clipboard to start.
  2. When filled with placeholders, click on one of the two buttons in each Maquette to specify which type you want to generate.
    "B" generates wide roads and "S" generates narrow/normal roads.
  3. The following actions can be done on each vehicle:
    • Left mouse button: enable/disable vehicle
    • Right mouse button: toggle between directions (left, right, straight)
    • Middle mouse button: edit vehicle title/name
  4. The following actions can be done on each corner:
    • Right mouse button: toggle between footpath, priority road
      or toggle the road between the current and the following corner (clockwise)
    • Left mouse button on signs to toggle them and right mouse button to toggle between different signs.
  5. Every Maquette has a number associated with it as the title/name.
    Normally it starts at 1 and counts upwards for each Maquette, but the counter can be restarted.
    To restart the title/name counter of a Maquette, right click on the type in the middle of the Maquette.
    The type is indicated by the letters "B" or "S".
    The restarting counter is highlighted with a color for ease (the color won't be printed).
  6. Every page has a top text and middle text indicated/enclosed by dashed lines.
    Left mouse button to change the text and shift + left mouse button to remove corresponding text.

PS: Disabled vehicles (indicated by a grey color) aren't printed.
